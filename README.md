# Praxis III PSA Code

This code implements the necessary functionality for the PSA assignment in Praxis III (ESC204) on a Raspberry Pi Pico board configured with CircuitPython. The functionality is as follows:

 - If button 1 is pressed, all lights turn off
 - If button 2 is pressed, only the red and blue lights turn on
 - If button 3 is pressed, all lights (green, red and blue) turn on

The default state is off.
